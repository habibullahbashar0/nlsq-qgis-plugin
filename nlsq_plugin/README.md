# NLSQ QGIS Plugin

Natural Language Spatial Query (NLSQ) allows you to perform spatial queries in QGIS using plain English.

## Installation

See [DEPLOYMENT.md](../DEPLOYMENT.md) for installation instructions.

## Usage

1. Click the NLSQ icon in the toolbar.
2. Enter your query (e.g., "Show parks within 500m of schools").
3. Click "Run".
